Historia Página y Sitio Web Elementos Tipos Creadores Gratuitos

Los elementos de una página web son los siguientes:

  • El texto: es el elemento más significativo de cualquier sitio Web porque los usuarios navegan por la Web fundamentalmente en busca de información expresada en hipertexto.
  • Las imágenes: constituyen un elemento esencial para ofrecer información visual del contenido y mostrar un diseño atractivo y personal.
  • Espacio Web: para que nuestro sitio Web sea visible ante todos necesitamos colocarlo en un servidor.
  • El nombre de dominio: es la primera parte de una dirección Web que identifica al servidor que almacena y sirve su sitio.
  • Hipervínculos: la relación de esta con una o más páginas por medio de enlaces.
  • Video: gracias al avance de las páginas web ahora es posible insertar videos en ellas.
  • Imagen Flash: Es una imagen animada, fácil de crear y sobre todo que ocupa poca memoria. Tiene diferentes usos, desde la creación de dibujos animados hasta el despliegue de dibujos interactivos.
  • Sonido: Un elemento cada vez más importante. La comercialización de nuevos dispositivos digitales móviles como el iPhone ha potencializado este elemento. Además de incorporar sonido en la página Web puede descargar de ellas archivos de audio para sus dispositivos móviles.
  • Banner: Una imagen fija o animada utilizada generalmente para publicidad.
  • Marcos: También llamados frames, son una especie de recuadro independiente en el que puede cargar una página Web.
