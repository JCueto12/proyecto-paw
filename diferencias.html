Historia Página y Sitio Web Elementos Tipos Creadores Gratuitos

Las Diferencia entre Página Web y Sitio Web son las siguientes


Pagina Web

Una página web es un documento o cualquier tipo de información que se crea en formato HTML, que es adaptado a la World Wide web y que se accede a su contenido a través de algunos de los navegadores existen. Se puede decir que las páginas Web son el soporte de la información en la Web.

Sitio Web

Un sitio Web es una localización en la Web que contiene varias páginas Web organizadas jerárquicamente. Cada sitio tiene una página de inicio que corresponde a la página principal, llamada en inglés Home Page. Un sitio tiene generalmente más de una página Web y su objetivo principal es ofrecer la información que necesitan los usuarios y proporcionársela de manera que puedan asignarla fácilmente.
